# frp-docker

Docker 部署 FRP 内网穿透服务

## 启动服务

```bash
# 启动 FRP Server 服务
sh ./scripts/frps/start.sh

# 启动 FRP Client 服务
sh ./scripts/frpc/start.sh
```

## 停止服务

```bash
# 停止 FRP Server 服务
sh ./scripts/frps/stop.sh
# 停止 FRP Client 服务
sh ./scripts/frpc/stop.sh
```

## 重启服务

```bash
# 重启 FRP Server 服务
sh ./scripts/frps/restart.sh
# 重启 FRP Client 服务
sh ./scripts/frpc/restart.sh
```
