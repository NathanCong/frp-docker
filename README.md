# frp-docker

Docker 部署 FRP 内网穿透服务

## 启动服务

```bash
# 启动 FRP Server 服务
sh ./scripts/start_frps.sh
# 启动 FRP Client 服务
sh ./scripts/start_frpc.sh
```

## 停止服务

```bash
# 停止 FRP Server 服务
sh ./scripts/stop_frps.sh
# 停止 FRP Client 服务
sh ./scripts/stop_frpc.sh
```

## 重启服务

```bash
# 重启 FRP Server 服务
sh ./scripts/restart_frps.sh
# 重启 FRP Client 服务
sh ./scripts/restart_frpc.sh
```
